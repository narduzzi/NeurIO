Machine Learning

NeurIO is an open source platform that provides tools to build, train and deploy machine learning models on embedded devices. Our platform provides end-to-end solutions for researchers to push the state-of-the-art in low-power ML, and build and deploy ML-powered applications for IoT.

What NeurIO offers

NeurIO provides benchmarking tools and an open source platform to deploy neural networks on embedded devices. Our platform is a comprehensive and flexible tool for researchers to push the state-of-the-art in low-power ML, and build and deploy ML-powered applications for IoT.

Benchmarking Tools

NeurIO provides benchmarking tools for researchers to compare the performance of different machine learning models on embedded devices.

Open source Platform

NeurIO provides an open source library in Python to optimize and deploy machine learning models on embedded devices.

Active Community

NeurIO is actively developed and maintained by important academic and industrial actors in the field of low-power machine learning.
